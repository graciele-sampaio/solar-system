## Baixando projeto:

Clone o repositório 
```bash
git@github.com:graciele-sampaio/solar-system.git
```

Entre na pasta criada:
```bash
cd solar-system
```

Instale as dependências
```bash
npm install
```

Para visualizar o projeto (irá rodar na porta 3000)
```bash
npm start
```
